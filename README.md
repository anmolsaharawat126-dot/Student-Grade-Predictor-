# Student Grade Predictor

## Overview
AI-powered student performance prediction system that estimates final grades based on academic indicators and attendance data.

## Features
- Grade Prediction
- Student Performance Analysis
- Attendance Impact Assessment
- Interactive Dashboard
- Machine Learning Based Prediction

## Tech Stack
- Python
- Pandas
- NumPy
- Scikit-Learn
- Streamlit
